# Icon System

Professional SVG icons for the Colomba Link website, replacing emoji-based icons for a more polished, professional appearance.

## Available Icons

### Services

- `icon-website.svg` - Global/website icon (🌐 replacement)
- `icon-webapp.svg` - Web applications icon (⚡ replacement)
- `icon-seo.svg` - Search/SEO icon (🔍 replacement)
- `icon-integration.svg` - Integration/connection icon (🔄 replacement)
- `icon-operations.svg` - Operations/deployment icon (🚀 replacement)

### Features

- `icon-performance.svg` - Performance/speed icon (⚡ replacement)
- `icon-maintenance.svg` - Maintenance/tools icon (🔧 replacement)
- `icon-connect.svg` - Integration/networking icon (🔗 replacement)
- `icon-speed.svg` - Speed/agility icon (🎯 replacement)

### Projects

- `icon-monitoring.svg` - Monitoring/analytics icon (📡 replacement)
- `icon-connect.svg` - People/connection icon (🤝 replacement)
- `icon-therapy.svg` - Healthcare/therapy icon (🏥 replacement)

## Design System

All icons follow a consistent design system:

- **Size**: 48x48px base size
- **Color**: Primary brand green (#6BA543)
- **Style**: Simple, geometric, with subtle opacity variations
- **Background**: Circular backdrop (20px radius) with 10% opacity
- **Line weight**: 2-2.5px for consistent visual weight

## Usage

Icons are displayed with styled containers in the pages:

```astro
<img src="/images/icon-name.svg" alt="Feature name" width="48" height="48" />
```

The icon containers have:

- Background gradients with brand colors
- Subtle borders
- Rounded corners (services/projects) or circles (features)
- Padding for visual breathing room
